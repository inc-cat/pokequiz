# PokéQuiz V1.0

See how many pokemon from each generation you can remember. Choose your settings at the top and type your guesses at the bottom!
Works for generation 1 to 9.
You can give up or reset at any time.

Enter `npm run start` to launch the game. The default page to view will be on [http://localhost:3000](http://localhost:3000)

Use `scripts/getPokemonData.js` to update Pokémon JSON data in `src/data`. All projects are commented.
